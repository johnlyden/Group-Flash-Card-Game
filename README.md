#JuJoJaTAY FLASH-FOO!

##Flash Card game built on Sinatra framework styled with Bootstrap

### Purpose
To test your knowlege on a number of cool flashcard decks.

### Installation
```
$ git clone [git-repo-url]
$ cd /repo
$ bundle
$ rake db:create
$ rake db:migrate
$ rake db:seed
$ be shotgun
```

### Technologies

*BCrypt
*Sinatra
